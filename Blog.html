What is the Document object Model (Dom)?


The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node in the tree corresponds to an element, attribute, or piece of text within the document.

What is event delegation in the context of the Dom,and why is it useful?


Event delegation is a technique in JavaScript that allows you to handle events at a higher level in the DOM (usually at a parent element) instead of attaching event listeners to specific child elements.
